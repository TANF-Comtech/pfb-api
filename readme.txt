pfb-api is the api project which provides all the workflow for grant applications used by the pfb application.

Dependencies used by this project
- Python 3.12
- uvicorn
- fastapi
- Django 5.0.3

- start command command prompt and run python -m uvicorn main:app
- to lanuch the api with command python -m uvicorn main:app --reload --port port#
- lanuch the api application (http://localhost:8000/docs)
